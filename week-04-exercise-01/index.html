My Dev Tools Vandalism

I used Chrome Developer Tools to temporarily edit the Expedia homepage. Since I enjoy traveling and have used Expedia to plan trips before, I thought it would be fun to personalize the page. I changed the main heading to "Ayse's Dream Destinations" and changed the text color, font size, and background color to make it stand out. It was interesting to see how quickly the page changed, even though none of the edits actually affected the real website.

What I "Vandalized"

I picked Expedia because it's a website I'm familiar with and have used when planning vacations. I wanted to make the homepage feel more personal, so I changed the main heading to "Ayse's Dream Destinations" and experimented with different colors and text sizes. This exercise made it easier to understand how HTML and CSS work together and showed me how useful Developer Tools can be for testing ideas before making permanent changes to a website.
